# Скиллы по ролям агентов

Один Claude Code инстанс можно превратить в специализированного агента, подобрав ему набор скиллов под конкретную задачу. Здесь — наша рабочая раскладка: какие роли мы используем в продакшене и какие скиллы каждой роли нужны.

Все скиллы лежат в `/skills/<name>/`. В этой папке — только описания ролей и индекс «кому что подключать».

## Роли

| Роль | Что делает | Папка |
|------|------------|-------|
| Marketing | Контент для соцсетей, посты, рилсы, YouTube, лендинги | [marketing-agent](./marketing-agent/) |
| Sales | Квалификация лидов, ресерч клиентов, follow-up, дожим | [sales-agent](./sales-agent/) |
| Coding | Большие задачи кода, MCP-серверы, ревью, DevOps | [coding-agent](./coding-agent/) |
| Coordinator | Оркестрация задач, делегирование, бэк-офис | [coordinator-agent](./coordinator-agent/) |
| Monitoring | Логи, метрики, чаты, daily digests | [monitoring-agent](./monitoring-agent/) |
| Reviewer | Code review, аудит решений, security | [reviewer-agent](./reviewer-agent/) |

## Как использовать

1. Выбери роль из списка выше
2. Открой её README — там список скиллов
3. Скопируй нужные скиллы из `/skills/` в `~/.claude/skills/`
4. Или клонируй весь репо целиком — Claude Code загружает все frontmatter'ы и сам подбирает по триггерам

## Зачем разделять

Один Claude Code может держать в контексте 30+ скиллов одновременно — frontmatter каждого скилла занимает ~100 слов. Но в рабочем сценарии разделение даёт три выгоды:

1. **Чёткие границы.** Marketing-агент не лезет в production-deploy. Coding-агент не пишет посты в Instagram.
2. **Композиция.** Можно запускать несколько Claude-инстансов параллельно с разными ролями (например через Telegram-боты или MCP-плагины).
3. **Понятная навигация.** Когда у тебя 100 скиллов в `~/.claude/skills/`, без ролевой карты ты не помнишь что у тебя есть.

## Универсальные скиллы

Эти скиллы полезны почти каждой роли:

- [`present`](../skills/present/) — HTML-презентации
- [`perplexity-research`](../skills/perplexity-research/) — web research
- [`markdown-new`](../skills/markdown-new/) — извлечение текста из URL

Поэтому они появляются в нескольких ролях ниже.
