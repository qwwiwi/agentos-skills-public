# YouTube Guide -- Сценарии, форматы, адаптация

> YouTube scripts проходят шаги 3-9 основного pipeline поста (SKILL.md §3).

## Format Matrix

| Формат | Хронометраж | Hook | Структура | CTA | Когда использовать |
|--------|------------|------|-----------|-----|--------------------|
| **Shorts** | 30-60 сек | 0-3 сек, одна фраза-провокация | 1 тезис + 1 пруф + вывод | Подписка / следующее видео | Горячая новость, один инсайт, мем-формат |
| **Короткое** | 5-7 мин | 0:00-0:30 | 2-3 блока, один аргумент каждый | Подписка + комментарий | Разбор одной темы, реакция на событие |
| **Среднее** | 15-20 мин | 0:00-0:45 | 4-6 блоков, нарастающая сложность | Подписка + ссылка на Telegram | Глубокий разбор, обзор рынка, кейс |
| **Длинное** | 30+ мин | 0:00-1:00, интрига или вопрос | 6-10 блоков, narrative arc | Подписка + Telegram + следующее видео | Образовательное, storytelling, интервью-формат |

## Структура сценария

### 1. Hook (первые 30 секунд -- решают всё)

Типы хуков:
- **Провокация:** «90% трейдеров теряют деньги. И вот почему это не изменится.»
- **Вопрос:** «Что будет с биткоином, если ФРС снизит ставку?»
- **Факт-удар:** «За последние 30 дней с бирж вывели 100K BTC. Это рекорд с 2022.»
- **Контр-нарратив:** «Все говорят альтсезон. Я думаю иначе.»

Правила хука:
- Без приветствий. Без «Привет, друзья». Сразу в тему.
- Одна мысль, одна интрига.
- Визуально: заголовок на экране совпадает с первой фразой.

### 2. Основная часть

Каждый блок:
```
[Тезис] -- одно предложение, что доказываем
[Аргумент] -- данные, график, пример
[Переход] -- мостик к следующему блоку или вопрос
```

Правила:
- 1 блок = 1 мысль. Не смешивать.
- Указывать в сценарии: `[ГРАФИК: BTC/USD дневной, показать уровень 65K]`
- Каждые 2-3 минуты -- retention hook: вопрос, тизер следующего блока, неожиданный факт.
- Категоричность принца сохраняется: «Большинство альтов умрёт» -- это стиль.

### 3. CTA (Call to Action)

- Органично в контексте, не «подпишитесь и нажмите колокольчик».
- Привязка к ценности: «Если хотите видеть такие разборы каждую неделю...»
- Telegram-ссылка: «Более детально -- в канале, ссылка в описании.»
- Один CTA -- один призыв. Не перегружать.

## Таймкоды и SEO-описание

### Таймкоды

Формат: `0:00 -- Название блока`

Пример для 15-минутного видео:
```
0:00 -- Биткоин на 70K: что дальше?
0:45 -- Что говорят объёмы
2:30 -- Funding rate и ликвидации
5:10 -- Макро: ФРС и инфляция
8:00 -- Два сценария на март
11:30 -- Моя позиция и уровни
14:00 -- Итоги и план
```

### SEO-описание

Структура:
```
[1-2 предложения -- суть видео, ключевое слово в первых 50 символах]

[Таймкоды]

[Полезные ссылки]
-- Telegram: [ссылка]
-- X/Twitter: [ссылка]

[Хэштеги -- 10-15 штук]
#bitcoin #btc #крипта #рынки #инвестиции #биткоин #трейдинг #макро #фрс #defi
```

### Теги

10-15 тегов. Микс:
- Широкие: bitcoin, crypto, инвестиции
- Средние: btc анализ, крипторынок 2025
- Узкие: биткоин март 2025, funding rate binance

## Адаптация Telegram → YouTube

| Аспект | Telegram | YouTube |
|--------|----------|--------|
| Длина | 500-1500 символов | 1000-5000 слов (сценарий) |
| Структура | Тезис → аргумент → P.S. | Hook → блоки → CTA |
| Визуал | Нет | Указания на графики, B-roll, текст на экране |
| Тон | Прямой, без вступлений | Прямой, но с retention hooks |
| Интерактив | Нет | «Напишите в комментариях», опросы |
| Данные | Компактно, 2-3 цифры | Развёрнуто, с контекстом |

### Процесс адаптации

1. **Выбери пост.** Лучше всего адаптируются посты с чёткой структурой «тезис + аргументы».
2. **Расширь каждый тезис** -- добавь контекст, пример, визуальное указание.
3. **Добавь hook** -- вынеси самый яркий тезис в начало.
4. **Retention hooks** -- вставь через каждые 2-3 минуты.
5. **CTA** -- привяжи к Telegram-каналу.
6. **Визуальные указания** -- `[ГРАФИК]`, `[СКРИНКАСТ]`, `[ТЕКСТ НА ЭКРАНЕ]`.

### Пример адаптации

**Telegram-пост (500 символов):**
> BTC пробил 70K. Объёмы x2 от среднего. Funding нейтральный -- рост здоровый, без leverage. Два сценария: 1) Консолидация 68-72K, потом 80K. 2) Откат к 65K на фиксации прибыли. Базовый -- первый. P.S. Смотрю за SPY -- если S&P покажет слабость, сценарий 2 станет вероятнее.

**YouTube-сценарий (outline):**
```
0:00 -- Hook: «BTC пробил 70K. Но вот что все упускают...» [ГРАФИК: BTC пробой]
0:30 -- Блок 1: Объёмы -- почему x2 это важно [ГРАФИК: объёмы, сравнение с октябрём]
2:00 -- Блок 2: Funding rate -- здоровый рост vs leverage [СКРИНКАСТ: Coinglass]
4:00 -- Блок 3: Два сценария + уровни [ГРАФИК: BTC с уровнями 65K/72K/80K]
6:00 -- Блок 4: Макро-фактор -- SPY и корреляция [ГРАФИК: BTC vs SPY]
8:00 -- Моя позиция + CTA → Telegram
```
