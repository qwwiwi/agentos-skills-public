# Datawrapper Chart Type IDs

## Bar charts
| ID | Description |
|----|-------------|
| d3-bars | Horizontal bars |
| d3-bars-stacked | Stacked bars |
| d3-bars-split | Split bars (grouped) |
| column-chart | Vertical bars (column) |

## Line / Area
| ID | Description |
|----|-------------|
| d3-lines | Line chart |
| d3-area | Area chart |

## Pie / Donut
| ID | Description |
|----|-------------|
| d3-pies | Pie chart |
| d3-donuts | Donut chart |

## Scatter / Dot
| ID | Description |
|----|-------------|
| d3-scatter-plot | Scatter plot |
| d3-dot-plot | Dot plot |

## Table
| ID | Description |
|----|-------------|
| tables | Responsive data table |

## Map (advanced)
| ID | Description |
|----|-------------|
| d3-maps-choropleth | Choropleth map |
| locator-map | Locator map |
| symbol-map | Symbol map |

## Best for financial data
- **Weekly purchases**: d3-bars (vertical) or column-chart
- **Price history**: d3-lines
- **Portfolio allocation**: d3-pies or d3-donuts
- **Comparison**: d3-bars-split
- **Volume + price overlay**: d3-lines (dual axis not native, use two series)
