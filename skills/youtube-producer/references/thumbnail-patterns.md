# Паттерны превью — Nate Herk

Превью ≈ 70% решения «кликать или нет». Nate Herk делает стабильно сильные превью по одной формуле – разберу её и дам шаблон под канал author.

## Композиция (одинакова в 80% видео)

```
┌───────────────────────────────────┐
│  [ЛЕВО: лицо]   [ПРАВО: объект] │
│   глаза в центр  логотип / скрин│
│                                    │
│    [КРУПНЫЙ ТЕКСТ внизу/сверху]   │
│     2–3 слова, CAPS, stroke       │
└───────────────────────────────────┘
```

## 6 обязательных элементов

1. **Лицо с эмоцией.** Не нейтральное – удивление, шок, недовольство, смех. Глаза смотрят в камеру либо на объект в превью.
2. **Крупный объект-триггер справа.** Логотип Claude, скриншот кода, цифра $10K, иконка OpenAI. Должно быть узнаваемо при 120px.
3. **Текст максимум 2–3 слова.** `DESTROYED`, `10x FASTER`, `$10K TRADER`, `99% CHEAPER`.
4. **Цвета:** тёмный фон + один акцентный (жёлтый, оранжевый, красный). Текст белый с чёрным stroke 4–6px.
5. **Стрелка или круг** вокруг главного элемента на скрине. Красная/жёлтая.
6. **Контраст:** лицо всегда светлее фона, объект справа в зоне «прокраста».

## Типы превью (4 формата)

### Тип 1 — Лицо + цифра (топ по CTR)
`I Turned Claude Into a 24/7 Trader` — слева злое лицо, справа график прибыли, текст `$10K TRADER` жёлтым.

### Тип 2 — Лицо + vs / против
`Claude Code vs Antigravity` — лицо в центре, слева логотип Claude, справа Antigravity, между ними красный «VS».

### Тип 3 — Лицо + «DESTROYED / KILLED»
`Claude Just Destroyed Every Video Editing Tool` — шокированное лицо слева, справа логотип Premiere/Final Cut с красным крестом, текст `DESTROYED` красным.

### Тип 4 — Скриншот-доказательство
`18 Token Hacks in 18 Minutes` — половина превью заполнена кодом/терминалом, текст `18 HACKS` поверх жёлтым кругом.

## Что НЕ делать

- Полный текст заголовка в превью – перегруз.
- Стоковые картинки и иконки без скрина/продукта.
- 3+ цвета текста – глаз не знает куда смотреть.
- Лицо без эмоции (нейтральное = невидимое).
- Мелкий текст – превью читают при 120px в мобиле.

## Шаблон промпта для генерации превью (ChatGPT Image 2 / Midjourney)

```
YouTube thumbnail. 1280x720. High contrast.
Left: close-up of [man / Dashi] with shocked/smirking expression, 
       eyes toward right side. Lit from front, dark background.
Right: [specific object: Claude logo crossed out / terminal with red error /
       dollar sign with arrow down].
Bottom center: bold CAPS text "[2–3 words]" in yellow/red, 
              thick black stroke 5px, slight tilt.
Red circle or arrow pointing at main object.
Mood: clickbait news, tech youtuber style, Nate Herk aesthetic.
```

## Channel adaptation

1. **Creator face required** на каждом превью (пока канал растёт – узнаваемость лица = subscribe).
2. Эмоции для ру-аудитории: удивление лучше злости. Ру-зритель скептичен к «злому клик-бейту».
3. Один акцентный цвет – **жёлтый** (сочетается с тёмным фоном, и не эмоционально нагружен как красный).
4. Русский текст в превью держать **одним словом или цифрой**: `СЛОМАЛ`, `ПОХОРОНИЛ`, `10x`, `$10K`, `99%`.
5. Для видео-релизов (тип A заголовков) – использовать логотип инструмента справа, лицо слева, `ПОХОРОНИЛ` посередине.
6. Для экспериментов (тип B) – скриншот результата + цифра прибыли/расхода.

## A/B тест превью

YouTube даёт нативный инструмент `Test & Compare` в Studio (до 3 превью).
Оставлять на тест минимум 48 часов. Не менять заголовок одновременно с превью.
