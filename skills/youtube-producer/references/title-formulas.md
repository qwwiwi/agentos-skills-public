# Формулы заголовков — Nate Herk (30 видео)

Средний: 116K views. Медиана: 106K. Топ: 397K.
Разбор: какая формула какие просмотры приносит + как адаптировать на русский.

---

## Формула 1 — AUTHORITY HIJACK (222K avg, топ)

Громкое имя + глагол-сила + инструмент. Создаёт ощущение новости об отрасли.

**Структура EN:** `[Known Name] Just [Power Verb] [Tool/Category]`

**Примеры:**
- "Andrej Karpathy Just 10x'd Everyone's Claude Code" — 397K
- "Claude Code + Paperclip Just Destroyed OpenClaw" — 222K
- "Claude + HeyGen Just Changed Content Creation Forever" — 45K

**RU адаптация:** `[Карпати / Альтман / Anthropic] только что [10x / сломал / похоронил] [Cursor / Claude Code / Copilot]`

**Примеры RU:**
- «Карпати только что в 10 раз прокачал Claude Code»
- «Anthropic только что похоронил ChatGPT для разработчиков»

**Акцент CAPS:** сильное слово — глагол или имя. `Карпати`, `10X`, `СЛОМАЛ`.

## Формула 2 — TRANSFORMATION / «I TURNED X INTO Y» (208K avg)

Личный опыт + превращение знакомого объекта в неожиданный результат.

**Структура EN:** `I Turned [Familiar] Into [Surprising Outcome]`

**Примеры:**
- "I Turned Claude Opus 4.7 Into a 24/7 Trader" — 208K
- "I Gave OpenClaw $10,000 to Trade Stocks" — 131K

**RU:** `Я превратил [Claude] в [24/7 трейдера]` / `Я дал [Claude] [$10k и попросил торговать]`

**Примеры RU:**
- «Я превратил Claude в 24/7 трейдера»
- «Я дал Claude $10 000 и он начал торговать сам»

**Акцент CAPS:** результат-существительное. `24/7 ТРЕЙДЕРА`, `$10 000`.

## Формула 3 — SHOCK NUMBER / «99% CHEAPER» (138K avg)

Конкретный процент или цифра + превосходная степень.

**Структура EN:** `[This/Tool] Just [Verb] [Tool] [Shock %] [Better/Cheaper/Faster]`

**Примеры:**
- "Ollama + Claude Code = 99% CHEAPER" — 207K
- "How to Use Claude Code for 99% CHEAPER" — 68K
- "18 Claude Code Token Hacks in 18 Minutes" — 189K
- "I've Built 500 AI Workflows, This is What Businesses Want" — 157K

**RU:** `[Инструмент] в 10 раз [дешевле/быстрее]` / `[N действий] за [N минут]` / `Я [собрал/протестировал] [N] [чего-то]`

**Примеры RU:**
- «Claude Code стал в 10 раз дешевле»
- «18 фишек Claude Code за 18 минут»
- «Я собрал 500 AI-воронок — вот что покупают в 2026»

**Акцент CAPS:** число. `99%`, `10 РАЗ`, `500`.

## Формула 4 — JUST DESTROYED / KILLED / BROKE (122K avg)

Глагол-разрушение + громкий субъект.

**Структура EN:** `[Subject] Just [Destroyed/Killed/Broke/Ended] [Category]`

**Примеры:**
- "Claude Just Destroyed Every Video Editing Tool" — 208K
- "Claude's New AI Just Changed the Internet Forever" — 205K
- "Gemini 3.1 Flash Live Just Changed Voice Agents Forever" — 27K

**RU:** `[Claude/Anthropic] только что [похоронил/убил/сломал] [Premiere / голосовых ботов / всю индустрию]`

**Примеры RU:**
- «Claude только что похоронил все редакторы видео»
- «Claude только что изменил интернет навсегда»
- «Gemini 3 только что уничтожил всех голосовых ботов»

**Акцент CAPS:** глагол. `ПОХОРОНИЛ`, `СЛОМАЛ`, `НАВСЕГДА`.

## Формула 5 — ABSOLUTISM / CONTRARIAN (128K avg)

Слова `Never / Forever / Everyone / Nobody / Stop` — обещание универсальности.

**Структура EN:** `You'll Never/Always [Action] [Again/The Same]` или `[Tool] Just Told Us To STOP [Action]`

**Примеры:**
- "Claude Just Told Us to Stop Using Their Best Model" — 118K
- "How to Never Hit Your Claude Session Limit Again" — 66K
- "Claude Code Source Code Just Leaked… 8 Things You Must Do" — 126K

**RU:** `Больше никогда не [делай X]` / `[Anthropic] попросили перестать использовать [Opus]` / `Все делают [X] неправильно`

**Примеры RU:**
- «Больше никогда не упирайся в лимиты Claude»
- «Anthropic попросили перестать использовать Opus»
- «Все используют Claude Code неправильно — вот 8 вещей»

**Акцент CAPS:** абсолютизм. `НИКОГДА`, `ВСЕ`, `ПЕРЕСТАНЬТЕ`.

## Что НЕ работает (антитоп)

| # | Views | Заголовок | Почему провал |
|---|-------|-----------|---------------|
| 1 | 20 720 | "OpenAI Image 2 is Nuts. Here are 10 Ways to Use it." | листикл «N ways» = признак AI-контента, скролл мимо |
| 2 | 24 507 | "Claude Code Just Gave Everyone Virtual Pets (April Fools?)" | вопрос в скобках = неуверенность, узкая тема |
| 3 | 27 708 | "Claude Design Builds Beautiful 3D Websites Instantly (full tutorial)" | слово «tutorial» = «будет скучно» |
| 4 | 45 889 | "Claude + HeyGen Just Changed Content Creation Forever" | узкая ниша (content creation) |
| 5 | 48 038 | "Seedance 2.0 + Claude Code = Beautiful $10k Websites" | абстрактная ценность «beautiful» |

**Правила-запреты:**
- Не ставь «N ways / N tips / N hacks» без жёсткой рамки времени («за 18 минут»).
- Не используй слова «tutorial», «guide», «how to» без шок-элемента.
- Не ставь вопрос в скобках — это сигнал неуверенности.
- Абстрактные прилагательные («beautiful», «amazing») ничего не весят.

## Что Nate ВСЕГДА делает

1. **Title Case** каждое слово (в русском — не работает так же, капитализируй триггерные слова).
2. Слово **`Just`** — создаёт ощущение news (17 из 30 видео). RU-аналог: **«только что»**.
3. **CAPS ключевого триггера:** `CHEAPER`, `DESTROYED`, `10x'd`.
4. Инструмент в заголовке в 87% видео (Claude / Claude Code / Cursor).
5. Первое лицо в personal-видео: `I Turned`, `I Gave`, `I Tested`.

## Эмфаза в русских заголовках

- Нельзя делать Title Case как в английском (выглядит как крик).
- Можно капитализировать **одно ключевое слово** в предложении: «Я ПОХОРОНИЛ Claude Code за 18 минут».
- Можно **выделить bold** одно число/действие в описании.
- Тире — только короткое –, не длинное —.
- Эмодзи — нет.

## Чек-лист RU-триггеров

**Глаголы-силы:** сломал, похоронил, убил, уничтожил, закрыл, прокачал, ускорил, удешевил, превратил.

**Абсолютизмы:** только что, впервые, навсегда, никогда, больше не, все, никто.

**Числа:** 10x, 99%, за N минут, $10 000, 500 штук, 3 секунды.

**Личное:** я дал, я превратил, я протестировал, я построил.

**Запреты:** tutorial, how to (в начале), guide, tips, ways (без рамки), amazing, beautiful.
